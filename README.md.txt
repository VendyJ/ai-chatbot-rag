# AI Chatbot s vyhledáváním v dokumentech (RAG)

Tento projekt ukazuje princip **Retrieval-Augmented Generation (RAG)**:  
Uživatel nahraje PDF dokument a chatbot odpovídá na otázky na základě jeho obsahu.
